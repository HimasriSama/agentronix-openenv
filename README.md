# Agentronix Email Triage Environment

This project implements a Reinforcement Learning environment for automated email triage.

The agent learns to classify incoming emails into categories such as:
- Work
- Personal
- Spam

Rewards are assigned based on correct categorization to help train reinforcement learning agents to manage email inboxes efficiently.

## Project Files

env.py – defines the email triage reinforcement learning environment  
tasks.py – contains sample email tasks for the agent  
grader.py – evaluates the agent’s performance and rewards  
baseline.py – baseline policy for testing the environment
